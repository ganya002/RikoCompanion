Nils Windows package

1. Copy this whole folder to the Windows PC.
2. Double-click Install-Nils-Windows.bat once.
3. After installation, start the app from the desktop shortcut "Nils Companion" or Run-Nils.vbs.

Notes:
- The installer creates a local .venv inside this folder.
- It installs CUDA-enabled PyTorch for NVIDIA GPUs.
- Voice cloning auto-transcription uses faster-whisper on Windows.
- The launcher uses pythonw, so it opens without a terminal window.
- Some macOS-only commands are limited on Windows, but chat, TTS, voice cloning, and screen capture are supported.
